TagMo is an amiibo™ manager and NFC reader / writer utility. Files used with TagMo are assumed the property of the user.

Please report any issues using the official TagMo issues page on GitHub. Issues reported elsewhere may not receive support.

TagMo is not affiliated, associated, authorized, endorsed by, or in any way officially connected with Nintendo. No amiibo™ or their digital representations are included in this project.

Binaries created with this project are not intended for sale or distribution. This project is for educational and archival purposes only.
